# Baking App

## Project Overview

Here is the Baking app project from Android Nanodegree course.

## Features

This app presents to the user an app with many recipes, which you can select it and see its ingredients and step-by-step to cook your favorite recipe!

## Libraries used

-   The following libraries were used for this project:
	- RecyclerView;
	- Picasso;
	- Glide:
	- Espresso
	- ExoPlayer;
	- ConstraintLayout;
	- Retrofit;
	- ButterKnife;
	- CardView;
	- Gson;
